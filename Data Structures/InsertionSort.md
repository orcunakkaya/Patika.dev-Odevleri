# Insertion Sort

## [22,27,16,2,18,6] -> Insertion Sort

1. Yukarı verilen dizinin sort türüne göre aşamalarını yazınız.
```
[22,27|16,2,18,6]
[16,22,27|2,18,6]
[2,16,22,27|18,6]
[2,16,18,22,27|6]
[2,6,16,18,22,27]
```

2. Big-O gösterimini yazınız.

```
O(n²)
```

3. Time Complexity

```

Worst case: O(n²)

```
